# FORM LAMARAN KERJA
Pada aplikasi ini terdapat beberapa fitur, diantaranya:
1.Pengisian Biodata : 
  * Nama
  * Alamat
  * Pendidikan Terakhir
  * Asal
  * Hobby
  
2.Pada pengisian **Nama** dan **Alamat** wajib diisi.

3.Menggunakan ScrollView.
